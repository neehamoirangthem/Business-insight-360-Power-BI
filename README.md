# Business-insight-360-Power-BI
## Project Overview
AtliQ Hardware is a global consumer electronics company. To support data-driven decision making across the organisation, this project delivers a comprehensive Power BI analytics solution that transform raw data into actionable insights.
The dashboard includes Finance, Sales, Marketing, Supply Chain and Executive views providing a unified analytical environment for stakeholders to monitor performance, identity trends and make strategic decisions with confidence.

### Features
 - Finance View: Comprehensive P&L dashboard visualizing Net Sales, GM% and Net Profit trends.
 - Sales View: Sales performance view combining customer metrics, product analysis and regional performance matrix.
 - Marketing View: Displaying segment-level sales, gross margin performance and unit economic trends.
 - Supply Chain View: Analyzing customer-wise forecast accuracy, net error trends, segment level risk.
 - Executive View: Combining financial KPIs, regional insights and competitive market share analysis.
